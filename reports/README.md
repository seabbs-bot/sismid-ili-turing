# Reports

One report per iteration loop of the model search.
Each report records what was tried, what the scores were, and what to try next,
so the search is auditable end to end.

Use [`TEMPLATE.md`](TEMPLATE.md) for each new loop.
Name reports `NN-short-slug.md` (e.g. `01-base-model.md`), numbered in order.
